# Client (Next.js app)

This folder is the **Attendance Agent** UI. Project overview, architecture, COE/session behaviour, and run instructions are in the **[repository root README](../README.md)**.

## Quick start

From this directory:

```bash
npm install
npm run dev
```

From the **repo root**, run `cd server` then `node index.js`. If you are already inside `server/`, use `node index.js` only. If the API is not on `http://localhost:5000`, set `NEXT_PUBLIC_API_URL` in `.env.local` (see root README).
